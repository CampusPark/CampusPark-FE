🖥 사용 방법 (Usage)

1. 실행 방법

```
# 클라이언트 실행
cd client
npm install
npm run dev

# 서버 실행
cd server
npm install
npm run start
```

2. 주요 기능
   • 지도에서 가까운 주차 공간 탐색
   • 시간대별 예약 및 결제
   • AI 기반 사진 검증
   • 마이페이지에서 신뢰도 점수 확인
